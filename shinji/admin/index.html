<a class="toc-item" href="#x1">短い表示名</a>
IDの付け方メモ
  • リンクで飛べるように、id は必須（例：t8 / r10 / h7 など）
  • 右目次の href は #id と一致させる
  • アンカーで飛んだ時に自動で開くスクリプトが入っていれば、追加してもそのまま効く

編集手順メモ

  1. 追加したい場所を決める（トップ / ルール / Q&A）
  2. テンプレをコピペして文章を置換
  3. id と右目次（toc-item）をセットで追加
  4. 表示崩れチェック（PC/スマホ幅）
  5. 公開する場合は Discord のリンク / 画像パスを最終確認
